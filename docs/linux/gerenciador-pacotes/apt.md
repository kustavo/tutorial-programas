# Apt

## Verificar a versão do pacote

Pacote que será instalado.

```bash
apt policy <nome-pacote>
```

Pacote instalado.

```bash
apt-cache policy <nome-pacote>
```

## Verificar dependências

Pacote que será instalado.

```bash
apt rdepends <nome-pacote>
```

Pacote instalado.

```bash
apt-cache rdepends <nome-pacote>
```
